# 中英生字翻譯系統

## 📚 系統簡介
這是一個完整的單機版中英生字翻譯系統，包含學生練習功能和管理者專頁，無需伺服器，可直接在瀏覽器中使用。

## 🚀 使用方法

### 學生模式
1. 直接用瀏覽器打開 `index.html` 文件
2. 選擇練習模式：
   - **中英翻譯**：看中文寫英文
   - **聽發音輸入英文**：聽發音寫英文
3. 點擊「開始練習」開始學習

### 管理者模式
1. 點擊「管理者模式」按鈕
2. 輸入帳號：`admin`，密碼：`admin@123`
3. 進入管理者專頁進行生字管理

## ✨ 功能特色

### 學生功能
- 🎯 **雙模式練習**：中英翻譯 + 聽發音練習
- 📊 **即時統計**：正確率、進度追蹤
- 🎧 **語音播放**：使用瀏覽器內建語音合成
- 📱 **響應式設計**：支援手機、平板、電腦
- ⌨️ **鍵盤支援**：Enter 鍵快速提交答案

### 管理者功能
- ➕ **單個新增**：添加新的中英生字
- 📝 **批量新增**：一次新增多個生字，支援批量導入
- ✏️ **編輯生字**：修改現有生字
- 🗑️ **刪除生字**：移除不需要的生字
- 💾 **資料儲存**：使用 localStorage 自動儲存
- 📋 **生字列表**：查看所有生字

## 🔐 管理者帳號
- **帳號**：admin
- **密碼**：admin@123

## 📝 操作說明

### 學生練習
- 選擇練習模式後點擊「開始練習」
- 輸入答案後按 Enter 鍵或點擊「提交答案」
- 查看結果後點擊「下一題」繼續
- 答對後可點擊「🔊 聽發音」聽到正確發音
- 練習完成後可選擇「重新練習」或「回到首頁」

### 管理者操作
- **單個新增**：在新增表單中輸入中文和英文，點擊「新增單個」
- **批量新增**：在批量新增區域輸入格式為「中文,英文」的生字，每行一個，點擊「批量新增」
- **編輯生字**：點擊生字旁的「編輯」按鈕，修改內容
- **刪除生字**：點擊生字旁的「刪除」按鈕，確認刪除
- **儲存變更**：點擊「儲存變更」將資料保存到瀏覽器

## 💡 技術特色
- **純前端**：無需伺服器，純 HTML/CSS/JavaScript，無任何依賴
- **無需安裝**：不需要 npm、node_modules 或其他構建工具
- **本地儲存**：使用 localStorage 保存生字資料
- **語音合成**：瀏覽器內建 Web Speech API
- **響應式設計**：適配各種螢幕尺寸
- **即時更新**：生字變更立即生效
- **GitHub Pages 友好**：可直接部署到 GitHub Pages，無需構建步驟

## 💡 注意事項
- 聽發音功能需要瀏覽器支援語音合成
- 建議使用 Chrome、Firefox、Edge 等現代瀏覽器
- 生字資料儲存在瀏覽器本地，清除瀏覽器資料會遺失
- 管理者帳號密碼已預設，可自行修改程式碼變更

## 🔧 自訂功能
如需修改系統設定，可編輯相關檔案中的變數：
- 修改預設生字：編輯 `index.html` 和 `admin.html` 中的 `words` 陣列
- 調整練習題數：修改 `index.html` 中的 `totalQuestions` 變數
- 更改管理者帳密：修改 `admin.html` 中的 `login()` 函數驗證邏輯
