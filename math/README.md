# 小學數學四則運算練習

## 專案概述

這是一個專為小學生設計的數學四則運算練習應用，提供多種練習模式和即時反饋。

## 功能特色

- **多種練習模式：** 加法、減法、乘法、除法、混合運算
- **九九乘法表練習：** 專項練習乘法口訣
- **進度追蹤：** 自動保存學習進度
- **即時反饋：** 答題後立即顯示結果
- **計時功能：** 練習時間限制
- **統計分析：** 答題正確率和速度統計

## 檔案結構

```
├── index.html          # 主頁面
├── script.js           # 主要邏輯
├── style.css           # 樣式檔案
├── admin.html          # 管理員頁面
├── package.json        # 專案配置
└── README.md           # 說明文件
```

## 使用方式

1. 直接在瀏覽器中開啟 `index.html`
2. 選擇練習類別
3. 開始答題練習
4. 查看統計結果

## 技術架構

- **前端：** HTML5, CSS3, JavaScript (ES6+)
- **打包：** Apache Cordova (Android APK)
- **存儲：** LocalStorage
