正則表達式

正則表達式用於匹配和處理文本模式。

基本用法

示例 1 import re

text = "Python is great"
pattern = r"Python"
match = re.search(pattern, text)
if match:
    print("找到匹配")

常見模式

示例 2 # 匹配數字
pattern = r"\d+"
text = "我有5個蘋果和3個香蕉"
numbers = re.findall(pattern, text)
print(numbers) # ['5', '3']

學習重點

  • 使用 re 模組
  • 正則表達式用於文本匹配
  • 常用模式：\d, \w, \s
  • findall 查找所有匹配
