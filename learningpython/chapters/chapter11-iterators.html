迭代器和生成器

迭代器和生成器是 Python 中高效處理數據流的重要工具。

迭代器

迭代器是一個可以記住遍歷位置的對象。

示例 1 my_list = [1, 2, 3]
my_iter = iter(my_list)

print(next(my_iter)) # 1
print(next(my_iter)) # 2
print(next(my_iter)) # 3

生成器

生成器是使用 yield 語句的特殊函數。

示例 2 def countdown(n):
    while n > 0:
        yield n
        n -= 1

for i in countdown(5):
    print(i)

生成器表達式

示例 3 squares = (x**2 for x in range(10))
for square in squares:
    print(square)

學習重點

  • 迭代器用於遍歷序列
  • 生成器節省內存
  • 使用 yield 創建生成器
  • 生成器表達式更簡潔
