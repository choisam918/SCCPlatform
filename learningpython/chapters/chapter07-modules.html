模組

模組是一個包含 Python 代碼的文件，用於組織和重用代碼。

導入模組

示例 1 import math
print(math.pi)
print(math.sqrt(16))

from...import

示例 2 from math import sqrt, pi
print(pi)
print(sqrt(25))

常用標準庫

示例 3 import random
print(random.randint(1, 10))

import datetime
now = datetime.datetime.now()
print(now)

學習重點

  • 模組用於組織和重用代碼
  • 使用 import 導入模組
  • Python 擁有豐富的標準庫
