Python 數據類型

變量是存儲在內存中的值，這就意味著在創建變量時會在內存中開闢一個空間。基於變量的數據類型，解釋器會分配指定內存，並決定什麼數據可以被存儲在內存中。因此，變量可以指定不同的數據類型，這些變量可以存儲整數，小數或字符。

變量賦值

Python 中的變量賦值不需要類型聲明。每個變量在內存中創建，都包括變量的標識，名稱和數據這些信息。每個變量在使用前都必須賦值，變量賦值以後該變量才會被創建。等號 = 用來給變量賦值。等號 = 運算符左邊是一個變量名，等號 = 運算符右邊是存儲在變量中的值。

示例 1 # 變量賦值示例
counter = 100 # 賦值整型變量
miles = 1000.0 # 浮點型
name = "John" # 字符串
print(counter)
print(miles)
print(name)
💡 輸出結果：
100
1000.0
John

多個變量賦值

Python 允許你同時為多個變量賦值。

示例 2 # 創建一個整型對象，值為 1，三個變量被分配到相同的內存空間上
a = b = c = 1
print(a, b, c) # 輸出：1 1 1

# 為多個對象指定多個變量
a, b, c = 1, 2, "john"
print(a, b, c) # 輸出：1 2 john

標準數據類型

在內存中存儲的數據可以有多種類型。例如，一個人的年齡可以用數字來存儲，他的名字可以用字符來存儲。Python 定義了一些標準類型，用於存儲各種類型的數據。Python 有五個標準的數據類型：

  • Numbers（數字）：int、float、complex
  • String（字符串）：str
  • List（列表）：list
  • Tuple（元組）：tuple
  • Dictionary（字典）：dict

Python 數字

數字數據類型用於存儲數值。他們是不可改變的數據類型，這意味著改變數字數據類型會分配一個新的對象。當你指定一個值時，Number 對象就會被創建。您也可以使用 del 語句刪除一些對象的引用。

示例 3 var1 = 1
var2 = 10

# 使用 del 語句刪除對象的引用
del var1
del var_a, var_b

Python 支持四種不同的數字類型：

  • int（有符號整型）
  • float（浮點型）
  • complex（複數）
數字類型實例 # int（有符號整型）
a = 10
b = -786
c = 0x69 # 十六進制

# float（浮點型）
x = 0.0
y = 15.20
z = -21.9
w = 32.3e+18 # 科學計數法

# complex（複數）
i = 3.14j
j = 45.j
k = 9.322e-36j
l = 3e+26J

print(type(a)) # <class 'int'>
print(type(y)) # <class 'float'>
print(type(i)) # <class 'complex'>
💡 提示：Python 還支持複數，複數由實數部分和虛數部分構成，可以用 a + bj 或者 complex(a,b) 表示，複數的實部 a 和虛部 b 都是浮點型。
⚠️ 注意：在 Python 3.x 版本中，long 類型被移除，使用 int 替代。long 類型只存在於 Python 2.x 版本中。

Python 字符串

字符串或串(String)是由數字、字母、下劃線組成的一串字符。一般記為：s = "a1a2···an" （n>=0）。它是編程語言中表示文本的數據類型。

python 的字串列表有 2 種取值順序：

  • 從左到右索引默認 0 開始，最大範圍是字符串長度少 1
  • 從右到左索引默認 -1 開始，最大範圍是字符串開頭

如果你要實現從字符串中獲取一段子字符串的話，可以使用 [頭下標:尾下標] 來截取相應的字符串，其中下標是從 0 開始算起，可以是正數或負數，下標可以為空表示取到頭或尾。[頭下標:尾下標] 獲取的子字符串包含頭下標的字符，但不包含尾下標的字符。

字符串示例 s = 'abcdef'
print(s[1:5]) # 輸出：bcde

# 字符串的基本操作
str = 'Hello World!'

print(str) # 輸出完整字符串：Hello World!
print(str[0]) # 輸出字符串中的第一個字符：H
print(str[2:5]) # 輸出字符串中第三個至第六個之間的字符串：llo
print(str[2:]) # 輸出從第三個字符開始的字符串：llo World!
print(str * 2) # 輸出字符串兩次：Hello World!Hello World!
print(str + "TEST") # 輸出連接的字符串：Hello World!TEST

# 字符串索引示例
print(str[-1]) # 輸出最後一個字符：!
print(str[-5:-1]) # 輸出：orld

Python 列表（List）

列表是 Python 中使用最頻繁的數據類型。列表可以完成大多數集合類的數據結構實現。列表中元素的類型可以不相同，它支持數字，字符串甚至可以包含列表（所謂嵌套）。

列表示例 list = ['runoob', 786, 2.23, 'john', 70.2]
tinylist = [123, 'john']

print(list) # 輸出完整列表：['runoob', 786, 2.23, 'john', 70.2]
print(list[0]) # 輸出列表的第一個元素：runoob
print(list[1:3]) # 輸出第二個至第三個元素：[786, 2.23]
print(list[2:]) # 輸出從第三個開始至列表末尾的所有元素：[2.23, 'john', 70.2]
print(tinylist * 2) # 輸出列表兩次：[123, 'john', 123, 'john']
print(list + tinylist) # 打印組合的列表：['runoob', 786, 2.23, 'john', 70.2, 123, 'john']

# 修改列表
list[0] = 'hello' # 修改第一個元素
list.append('world') # 在列表末尾添加元素
print(list) # ['hello', 786, 2.23, 'john', 70.2, 'world']
💡 提示：列表的索引從 0 開始，可以使用負數索引從末尾開始訪問。列表是可變的，可以修改、添加、刪除元素。

Python 元組（Tuple）

元組（tuple）與列表類似，不同之處在於元組的元素不能修改。元組寫在小括號 () 裡，元素之間用逗號隔開。元組中的元素類型也可以不相同。

元組示例 tuple = ('runoob', 786, 2.23, 'john', 70.2)
tinytuple = (123, 'john')

print(tuple) # 輸出完整元組：('runoob', 786, 2.23, 'john', 70.2)
print(tuple[0]) # 輸出元組的第一個元素：runoob
print(tuple[1:3]) # 輸出第二個至第三個元素：(786, 2.23)
print(tuple[2:]) # 輸出從第三個開始至列表末尾的所有元素：(2.23, 'john', 70.2)
print(tinytuple * 2) # 輸出元組兩次：(123, 'john', 123, 'john')
print(tuple + tinytuple) # 打印組合的元組：('runoob', 786, 2.23, 'john', 70.2, 123, 'john')

# 元組與字符串類似，可以被索引且下標索引從0開始，-1 為從末尾開始的位置。
# tuple[0] = 'hello' # 錯誤！元組不能修改

# 創建空元組
tup1 = ()
# 單個元素的元組需要逗號
tup2 = (50,)
⚠️ 注意：元組中的元素不能修改，這是元組與列表的主要區別。雖然元組的元素不可改變，但它可以包含可變的對象，比如列表。

Python 字典（Dictionary）

字典（dictionary）是 Python 中另一個非常有用的內置數據類型。字典是一種映射類型（mapping type），它是一個無序的鍵 : 值對集合。關鍵字必須使用不可變類型，也就是說列表和包含可變類型的元組不能用作關鍵字。

字典示例 dict = {}
dict['one'] = "This is one"
dict[2] = "This is two"

tinydict = {'name': 'john', 'code': 6734, 'dept': 'sales'}

print(dict['one']) # 輸出鍵為 'one' 的值：This is one
print(dict[2]) # 輸出鍵為 2 的值：This is two
print(tinydict) # 輸出完整的字典：{'name': 'john', 'code': 6734, 'dept': 'sales'}
print(tinydict.keys()) # 輸出所有鍵：dict_keys(['name', 'code', 'dept'])
print(tinydict.values()) # 輸出所有值：dict_values(['john', 6734, 'sales'])

# 修改字典
dict['one'] = "Updated one" # 修改現有鍵的值
dict['three'] = "This is three" # 添加新的鍵值對
print(dict)

# 刪除鍵值對
del dict['one'] # 刪除鍵 'one'
# dict.clear() # 清空字典
# del dict # 刪除字典
💡 提示：字典是一種映射類型，它的元素是鍵值對。字典的鍵必須是不可變類型（如字符串、數字、元組），且不能重複。

集合（Set）

集合是一個無序的、不重複的元素集合，用花括號 {} 表示。

示例 7 # 集合示例
numbers = {1, 2, 3, 4, 5}
letters = set('hello') # 創建字符集合

print(numbers) # 輸出：{1, 2, 3, 4, 5}
print(letters) # 輸出：{'o', 'h', 'e', 'l'}

# 集合操作
numbers.add(6) # 添加元素
numbers.discard(3) # 刪除元素
print(numbers) # 輸出：{1, 2, 4, 5, 6}

布爾類型（Boolean）

布爾類型只有兩個值：True 和 False。

示例 8 # 布爾類型示例
is_student = True
is_working = False

print(type(is_student)) # 輸出：<class 'bool'>
print(10 > 5) # 輸出：True
print(10 < 5) # 輸出：False
⚠️ 注意：布爾值的大小寫很重要，True 和 False 必須首字母大寫。

類型轉換

Python 提供了內置函數來進行類型轉換。

示例 9 # 類型轉換
x = "123"
y = int(x) # 轉換為整數

a = 3.14
b = int(a) # 轉換為整數（會截斷小數部分）

number = 42
text = str(number) # 轉換為字符串

print(type(y)) # 輸出：<class 'int'>
print(type(text)) # 輸出：<class 'str'>

學習重點

  • Python 有多種內置數據類型，每種都有其特定的用途
  • 列表是可變的有序序列，用方括號 [] 表示
  • 元組是不可變的有序序列，用圓括號 () 表示
  • 字典是鍵值對的集合，用花括號 {} 表示
  • 可以使用 type() 函數查看變量的類型
  • 可以使用各種內置函數進行類型轉換
