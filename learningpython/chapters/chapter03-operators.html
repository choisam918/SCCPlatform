Python 運算符

運算符用於對變量和值進行操作。Python 提供了多種運算符，包括算術運算符、比較運算符、邏輯運算符、賦值運算符等。

算術運算符

用於執行基本的數學運算。

示例 1 # 算術運算符
a = 10
b = 3

print(a + b) # 加法：13
print(a - b) # 減法：7
print(a * b) # 乘法：30
print(a / b) # 除法：3.333...
print(a % b) # 取餘：1
print(a ** b) # 次方：1000
print(a // b) # 整除：3

比較運算符

用於比較兩個值，返回布爾值。

示例 2 # 比較運算符
x = 5
y = 10

print(x == y) # 等於：False
print(x != y) # 不等於：True
print(x > y) # 大於：False
print(x < y) # 小於：True
print(x >= y) # 大於等於：False
print(x <= y) # 小於等於：True

賦值運算符

用於給變量賦值。

示例 3 # 賦值運算符
x = 10
x += 5 # 等同於 x = x + 5
print(x) # 15

x -= 3 # 等同於 x = x - 3
print(x) # 12

x *= 2 # 等同於 x = x * 2
print(x) # 24

x /= 4 # 等同於 x = x / 4
print(x) # 6.0

邏輯運算符

用於組合條件語句。

示例 4 # 邏輯運算符
a = True
b = False

print(a and b) # 邏輯與：False
print(a or b) # 邏輯或：True
print(not a) # 邏輯非：False
💡 提示：邏輯運算符常用於條件語句中，用於組合多個條件。

成員運算符

用於檢查值是否屬於某個序列。

示例 5 # 成員運算符
fruits = ['apple', 'banana', 'orange']

print('apple' in fruits) # True
print('grape' not in fruits) # True

身份運算符

用於比較兩個對象是否為同一個對象。

示例 6 # 身份運算符
a = [1, 2, 3]
b = a
c = [1, 2, 3]

print(a is b) # True (同一個對象)
print(a is c) # False (不同對象)
print(a == c) # True (值相同)

運算符優先級

Python 運算符有特定的優先級順序。

示例 7 # 運算符優先級
result = 2 + 3 * 4
print(result) # 14 (先乘法後加法)

result = (2 + 3) * 4
print(result) # 20 (括號優先)

學習重點

  • 算術運算符用於數學計算
  • 比較運算符返回布爾值
  • 賦值運算符用於簡化賦值操作
  • 邏輯運算符用於組合多個條件
  • 成員運算符用於檢查元素是否在序列中
  • 運算符有優先級順序，可以用括號改變
