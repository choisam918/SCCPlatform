循環語句

循環語句允許程序重複執行一段代碼。Python 提供了兩種主要的循環結構：for 循環和 while 循環。

for 循環

for 循環用於遍歷序列或其他可迭代對象。

示例 1 fruits = ['apple', 'banana', 'orange']
for fruit in fruits:
    print(fruit)

range() 函數

range() 函數用於生成一個數字序列。

示例 2 for i in range(5):
    print(i)

while 循環

while 循環會在條件為真時持續執行。

示例 3 count = 0
while count < 5:
    print(count)
    count += 1

循環控制語句

示例 4 for i in range(10):
    if i == 5:
        break
    print(i)

學習重點

  • for 循環用於遍歷序列
  • while 循環在條件為真時執行
  • break 用於跳出循環
  • continue 用於跳過本次循環
