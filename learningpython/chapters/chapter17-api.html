API 開發

Python 可以使用 Flask 或 FastAPI 來開發 RESTful API。

使用 Flask

示例 1 from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/api/users', methods=['GET'])
def get_users():
    users = [{'id': 1, 'name': '張三'}]
    return jsonify(users)

if __name__ == '__main__':
    app.run(debug=True)

使用 FastAPI

示例 2 from fastapi import FastAPI

app = FastAPI()

@app.get("/api/users")
def get_users():
    return [{"id": 1, "name": "張三"}]

學習重點

  • Flask 是輕量級框架
  • FastAPI 性能更好
  • 使用路由處理 HTTP 請求
  • 返回 JSON 格式數據
