裝飾器

裝飾器是修改或擴展函數而不修改函數本身的一種方式。

基本裝飾器

示例 1 def my_decorator(func):
    def wrapper():
        print("裝飾器執行前")
        func()
        print("裝飾器執行後")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()

帶參數的裝飾器

示例 2 def timer_decorator(func):
    import time
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"執行時間: {end - start} 秒")
        return result
    return wrapper

@timer_decorator
def slow_function():
    time.sleep(1)
    return "完成"

學習重點

  • 裝飾器用於修改函數行為
  • 使用 @ 語法應用裝飾器
  • 可以傳遞參數
  • 裝飾器讓代碼更簡潔
