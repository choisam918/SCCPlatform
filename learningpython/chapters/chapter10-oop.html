面向對象編程

面向對象編程（OOP）是一種編程範式，使用"對象"來設計應用程序。

類和對象

示例 1 class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        print(f"{self.name} 在叫")

dog1 = Dog("小黑")
dog1.bark()

繼承

示例 2 class Animal:
    def __init__(self, name):
        self.name = name

class Dog(Animal):
    def bark(self):
        print("汪汪！")

學習重點

  • 類是對象的藍圖
  • 對象是類的實例
  • 使用 __init__ 初始化
  • 繼承可以重用代碼
