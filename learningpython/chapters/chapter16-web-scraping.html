網絡爬蟲

網絡爬蟲用於從網站上抓取數據。Python 常用的庫有 requests 和 BeautifulSoup。

使用 requests

示例 1 import requests

response = requests.get('https://www.example.com')
print(response.status_code)
print(response.text)

使用 BeautifulSoup

示例 2 from bs4 import BeautifulSoup
import requests

response = requests.get('https://www.example.com')
soup = BeautifulSoup(response.text, 'html.parser')

# 查找標題
title = soup.find('title')
print(title.text)
注意：使用網絡爬蟲時要遵守網站的 robots.txt 和使用條款，不要過度請求。

學習重點

  • 使用 requests 發送 HTTP 請求
  • 使用 BeautifulSoup 解析 HTML
  • 遵守網站的使用條款
  • 注意請求頻率，避免過度負載
