文件操作

文件操作是程序處理數據的重要方式。

打開文件

示例 1 file = open('test.txt', 'r')
content = file.read()
file.close()

with 語句

示例 2 with open('test.txt', 'r') as file:
    content = file.read()
    print(content)

寫入文件

示例 3 with open('output.txt', 'w') as file:
    file.write('Hello, World!')

學習重點

  • 使用 open() 打開文件
  • 使用 with 語句自動關閉文件
  • 'r' 為讀取，'w' 為寫入
