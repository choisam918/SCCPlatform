數據庫操作

Python 可以連接和操作各種數據庫，常用的有 SQLite、MySQL、PostgreSQL 等。

SQLite

示例 1 import sqlite3

# 連接數據庫
conn = sqlite3.connect('example.db')
cursor = conn.cursor()

# 創建表
cursor.execute('''CREATE TABLE users
              (id INTEGER PRIMARY KEY, name TEXT)''')

# 插入數據
cursor.execute("INSERT INTO users (name) VALUES (?)", ("張三",))
conn.commit()
conn.close()

查詢數據

示例 2 conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute("SELECT * FROM users")
rows = cursor.fetchall()

for row in rows:
    print(row)

conn.close()

學習重點

  • 使用 sqlite3 模組
  • connect() 連接數據庫
  • execute() 執行 SQL
  • commit() 提交事務
