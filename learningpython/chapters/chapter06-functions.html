函數

函數是一段可重複使用的代碼塊，用於執行特定的任務。函數可以接受輸入參數，也可以返回結果。Python 中有內置函數，也可以自定義函數。

定義函數

使用 def 關鍵字來定義函數。

示例 1 # 定義一個簡單的函數
def greet():
    print("Hello, World!")

# 調用函數
greet() # 輸出：Hello, World!

帶參數的函數

函數可以接受參數。

示例 2 # 帶參數的函數
def greet(name):
    print(f"Hello, {name}!")

greet("張三") # 輸出：Hello, 張三!
greet("李四") # 輸出：Hello, 李四!

多個參數

函數可以接受多個參數。

示例 3 # 多個參數
def add(x, y):
    result = x + y
    return result

sum = add(3, 5)
print(sum) # 輸出：8

默認參數

可以為函數參數設置默認值。

示例 4 # 默認參數
def introduce(name, age=18, city="台北"):
    print(f"我是{name}，{age}歲，來自{city}")

introduce("張三") # 使用默認參數
# 輸出：我是張三，18歲，來自台北

introduce("李四", 25, "台中") # 提供所有參數
# 輸出：我是李四，25歲，來自台中

關鍵字參數

可以通過參數名來傳遞參數。

示例 5 # 關鍵字參數
def create_profile(name, age, occupation):
    print(f"姓名：{name}")
    print(f"年齡：{age}")
    print(f"職業：{occupation}")

# 使用關鍵字參數，順序可以改變
create_profile(age=25, name="王五", occupation="工程師")

可變參數

使用 *args 和 **kwargs 接受可變數量的參數。

示例 6 # *args 接受任意數量的位置參數
def sum_all(*args):
    total = 0
    for num in args:
        total += num
    return total

print(sum_all(1, 2, 3)) # 輸出：6
print(sum_all(1, 2, 3, 4, 5)) # 輸出：15
💡 提示：使用可變參數可以使函數更加靈活，可以處理不同數量的輸入。

返回值

使用 return 關鍵字返回函數結果。

示例 7 # 返回多個值
def calculate(x, y):
    add = x + y
    multiply = x * y
    return add, multiply

result = calculate(3, 4)
print(result) # 輸出：(7, 12)

# 解包返回值
sum, product = calculate(5, 6)
print(sum) # 輸出：11
print(product) # 輸出：30

局部變量和全局變量

函數內部定義的變量是局部變量，只在函數內部有效。

示例 8 # 局部變量和全局變量
x = 10 # 全局變量

def test():
    x = 20 # 局部變量
    print(f"函數內部 x = {x}")

test() # 輸出：函數內部 x = 20
print(f"函數外部 x = {x}") # 輸出：函數外部 x = 10

def test_global():
    global x # 聲明使用全局變量
    x = 30

test_global()
print(f"修改後 x = {x}") # 輸出：修改後 x = 30
⚠️ 注意：在函數內部修改全局變量需要使用 global 關鍵字。

lambda 函數

lambda 函數是一種匿名的簡短函數。

示例 9 # lambda 函數
multiply = lambda x, y: x * y
print(multiply(3, 4)) # 輸出：12

# 與普通函數等價
def multiply_normal(x, y):
    return x * y

學習重點

  • 使用 def 關鍵字定義函數
  • 函數可以接受參數並返回值
  • 可以為參數設置默認值
  • 使用 *args 和 **kwargs 處理可變參數
  • 局部變量只在函數內部有效
  • 使用 global 關鍵字修改全局變量
  • lambda 函數用於創建簡單的匿名函數
