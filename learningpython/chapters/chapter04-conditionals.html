條件語句

條件語句允許程序根據不同的條件執行不同的代碼塊。Python 使用 if、elif 和 else 關鍵字來構建條件語句。

if 語句

最簡單的條件語句是 if 語句，當條件為真時執行代碼塊。

示例 1 # 基本 if 語句
age = 18
if age >= 18:
    print("您已成年")

if-else 語句

當條件為真時執行一個代碼塊，條件為假時執行另一個代碼塊。

示例 2 exporting
age = 15
if age >= 18:
    print("您已成年")
else:
    print("您未成年")

if-elif-else 語句

用於檢查多個條件。

示例 3 # 多條件判斷
score = 85

if score >= 90:
    print("優秀")
elif score >= 80:
    print("良好")
elif score >= 70:
    print("中等")
else:
    print("需要努力")

嵌套條件語句

可以在 if 語句內部再放置另一個 if 語句。

示例 4 # 嵌套條件語句
age = 20
has_license = True

if age >= 18:
    if has_license:
        print("可以開車")
    else:
        print("需要先考駕照")
else:
    print("年齡不足")
⚠️ 注意：Python 的縮進非常重要，錯誤的縮進會導致邏輯錯誤。

邏輯運算符

可以結合使用 and、or、not 來構建複雜的條件。

示例 5 # 使用邏輯運算符
age = 20
has_license = True
has_car = False

if age >= 18 and has_license:
    print("可以開車")

if has_car or has_license:
    print("有交通工具")

if not has_car:
    print("沒有汽車")
💡 提示：使用邏輯運算符可以讓條件判斷更加簡潔清晰。

三元運算符

Python 提供了簡短的三元運算符語法。

示例 6 # 三元運算符
age = 20
status = "成年" if age >= 18 else "未成年"
print(status) # 輸出：成年

# 等價於：
if age >= 18:
    status = "成年"
else:
    status = "未成年"

學習重點

  • if 語句用於檢查單個條件
  • if-else 語句用於處理兩種情況
  • if-elif-else 語句用於檢查多個條件
  • 可以使用邏輯運算符 and、or、not
  • 條件語句可以嵌套使用
  • Python 的三元運算符提供簡潔的條件賦值語法
