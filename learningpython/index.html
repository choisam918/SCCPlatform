Python 基本語法

Python 是一種簡單易學的編程語言，具有清晰的語法結構。本章節將介紹 Python 的基本語法規則，包括縮進、註釋、變量命名等基礎知識。

Python 縮進

與其他編程語言（如 C++ 或 Java）不同，Python 使用縮進來表示代碼塊，而不是使用大括號。這是 Python 最顯著的特點之一。

示例 1 
if True:
    print("這是縮進的代碼塊")
    print("這一行也屬於同一個代碼塊")
print("這一行不在條件語句內")
⚠️ 注意：縮進必須一致！如果一個代碼塊使用 4 個空格，則整個代碼塊都應該使用 4 個空格。不要混用空格和製表符（Tab）。

Python 註釋

註釋用於解釋代碼的功能，不會被 Python 解釋器執行。Python 支持單行註釋和多行註釋。

單行註釋

使用 # 符號開始單行註釋。

示例 2 
# 這是一個單行註釋
print("Hello, World!")  # 這是行尾註釋

多行註釋

使用三個引號 """ 或 ''' 來創建多行註釋。

示例 3 
"""
這是一個多行註釋
可以寫多行內容
用於文檔說明
"""

Python 多行語句

Python 通常一行寫完一個語句，但如果語句太長，可以使用反斜杠 \ 來分行。

示例 4 
total = item_one + \
        item_two + \
        item_three
💡 提示：在列表、元組或字典中，Python 會自動識別多行，不需要使用反斜杠。

Python 引號

Python 可以使用單引號 '、雙引號 " 和三引號 """ 來表示字符串。

示例 5 
word = 'word'
sentence = "這是一個句子"
paragraph = """這是一個段落，
可以包含多行，
非常方便！"""

Python 變量命名規則

  • 變量名必須以字母（a-z, A-Z）或下劃線 _ 開頭
  • 變量名可以包含字母、數字和下劃線
  • 變量名是區分大小寫的（如 name 和 Name 是不同的變量）
  • 不能使用 Python 的保留字作為變量名
示例 6 
# 有效的變量名
name = "張三"
age = 25
_is_student = True
student_name = "李四"

# 無效的變量名
# 2name = "王五"  # 不能以數字開頭
# my-name = "趙六"  # 不能包含連字符
# if = 10  # 不能使用保留字
💡 命名建議：使用有意義的變量名，遵循命名慣例（如使用下劃線分隔單詞，類名使用駝峰命名法等）。

Python 保留字

以下是在 Python 中有特殊意義的保留字，不能用作變量名：

保留字列表（Python 3） 
False  None  True  and  as  assert  async  await
break  class  continue  def  del  elif  else  except
finally  for  from  global  if  import  in  is  lambda
nonlocal  not  or  pass  raise  return  try  while  with
yield  match  case

學習重點

  • Python 使用縮進而不是大括號來定義代碼塊
  • 使用 # 進行單行註釋，使用三引號進行多行註釋
  • 變量名必須遵循命名規則，不能使用保留字
  • 字符串可以使用單引號、雙引號或三引號
  • 保持代碼整潔和一致的縮進風格非常重要
