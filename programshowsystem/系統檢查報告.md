## ⚠️ 已知限制

### 1. Pyodide 載入
- 首次載入需要 10-30 秒（依網路速度）
- 需要網路連線（從 CDN 下載約 10MB）
- 某些 Python 功能不支援（檔案操作、網路請求等）

### 2. 儲存限制
- LocalStorage 約 5-10MB（依瀏覽器而定）
- 單檔最大 1MB

### 3. 瀏覽器相容性
- 需要支援 WebAssembly
- 需要支援 LocalStorage 和 FileReader API
