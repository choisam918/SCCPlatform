# 程式展示系統 - 使用說明

## 🚀 快速開始

### 本地使用

1. **下載或克隆專案**
   ```bash
   git clone https://github.com/yourusername/programshowsystem.git
   cd programshowsystem
   ```

2. **開啟系統**
   - 直接用瀏覽器開啟 `index.html`
   - 或使用本地伺服器（推薦）：
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Node.js
     npx http-server
     ```
     然後訪問 `http://localhost:8000`

### GitHub Pages 部署

1. **推送到 GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/yourusername/programshowsystem.git
   git push -u origin main
   ```

2. **啟用 GitHub Pages**
   - 進入 Repository Settings
   - 找到 Pages 設定
   - Source 選擇 `main` 分支
   - Folder 選擇 `/ (root)`
   - 儲存後等待幾分鐘

3. **訪問網站**
   - URL: `https://yourusername.github.io/programshowsystem/`

## 📖 功能說明

### 1. 上傳檔案

- **方式1**：點擊「選擇檔案」按鈕
- **方式2**：拖放檔案到上傳區域
- **支援**：多檔上傳（按住 Ctrl/Cmd 選擇多個檔案）
- **限制**：僅支援 `.py` 檔案，單檔最大 1MB

### 2. 檔案管理

- **預覽**：點擊「預覽」按鈕查看檔案內容
- **刪除**：點擊「刪除」按鈕移除檔案
- **清除全部**：一次清除所有檔案
- **匯出資料**：將所有檔案匯出為 JSON 格式
- **匯入資料**：從 JSON 檔案匯入之前匯出的資料

### 3. 生成展示

- 點擊「生成展示頁面」按鈕
- 系統會自動跳轉到展示頁面
- 所有上傳的 Python 檔案會以語法高亮方式顯示

### 4. 展示頁面功能

- **返回上傳**：回到上傳頁面
- **列印**：列印當前展示內容
- **匯出HTML**：將展示內容匯出為獨立的 HTML 檔案
- **切換主題**：在淺色/深色主題間切換
- **執行程式碼**：在瀏覽器中直接執行 Python 程式碼並顯示結果 ⭐ 新功能

## 💾 資料儲存

### LocalStorage

- 所有檔案資料儲存在瀏覽器的 LocalStorage 中
- **容量限制**：約 5-10MB（依瀏覽器而定）
- **持久性**：資料會保留在瀏覽器中，直到清除瀏覽器資料

### 備份建議

- 定期使用「匯出資料」功能備份
- 匯出的 JSON 檔案可以隨時匯入恢復

## ⚠️ 注意事項

1. **瀏覽器相容性**
   - 建議使用現代瀏覽器（Chrome、Firefox、Safari、Edge）
   - 需要支援 LocalStorage 和 FileReader API

2. **檔案大小限制**
   - 單檔最大 1MB
   - 總儲存空間約 5MB（依瀏覽器而定）

3. **資料安全**
   - 所有資料僅存在本地瀏覽器
   - 清除瀏覽器資料會導致檔案遺失
   - 建議定期匯出備份

4. **隱私保護**
   - 系統完全在本地運行
   - 不會上傳任何資料到伺服器
   - 適合處理敏感資料

## 🔧 故障排除

### 問題：無法上傳檔案

**解決方案**：
- 確認檔案是 `.py` 格式
- 確認檔案大小不超過 1MB
- 檢查瀏覽器控制台是否有錯誤訊息

### 問題：儲存空間不足

**解決方案**：
- 刪除不需要的檔案
- 匯出資料後清除全部
- 使用「匯出HTML」功能保存展示結果

### 問題：展示頁面沒有語法高亮

**解決方案**：
- 確認網路連線（使用CDN版本）
- 檢查瀏覽器控制台是否有錯誤
- 重新整理頁面

### 問題：無法執行程式碼

**解決方案**：
- 確認網路連線正常（需要載入 Pyodide，約 10MB）
- 首次執行需要載入時間（約 10-30 秒）
- 確認瀏覽器支援 WebAssembly
- 檢查程式碼是否有語法錯誤
- 某些 Python 功能可能不支援（如檔案操作、網路請求等）

### 問題：資料遺失

**解決方案**：
- 檢查是否清除了瀏覽器資料
- 如果有匯出備份，使用「匯入資料」恢復
- 下次記得定期備份

## 📝 技術細節

### 使用的技術

- **HTML5**：頁面結構
- **CSS3**：樣式和響應式設計
- **JavaScript (ES6+)**：核心功能
- **LocalStorage API**：資料儲存
- **FileReader API**：檔案讀取
- **highlight.js**：程式碼語法高亮
- **Pyodide**：在瀏覽器中執行 Python 程式碼 ⭐

### 瀏覽器支援

- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+
