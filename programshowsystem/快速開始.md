# 🚀 快速開始指南

## 第一步：測試系統

### 方式1：直接開啟（最簡單）

1. 找到 `index.html` 檔案
2. 雙擊或用瀏覽器開啟
3. 開始使用！

### 方式2：使用本地伺服器（推薦）

```bash
# 在專案目錄下執行

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx http-server

# 然後訪問 http://localhost:8000
```

## 第二步：測試功能

1. **上傳測試檔案**
   - 使用 `examples/` 目錄中的範例檔案
   - 或上傳您自己的 Python 檔案

2. **測試各項功能**
   - ✅ 檔案上傳（拖放或點擊）
   - ✅ 檔案預覽
   - ✅ 檔案刪除
   - ✅ 生成展示頁面
   - ✅ 語法高亮顯示
   - ✅ 匯出功能

## 第三步：部署到 GitHub

### 1. 建立 GitHub Repository

```bash
# 初始化 Git
git init

# 添加所有檔案
git add .

# 提交
git commit -m "Initial commit: 程式展示系統"

# 在 GitHub 上建立新的 Repository
# 然後連接遠端
git remote add origin https://github.com/yourusername/programshowsystem.git

# 推送
git push -u origin main
```

### 2. 啟用 GitHub Pages

1. 進入 GitHub Repository
2. 點擊 **Settings**
3. 左側選單找到 **Pages**
4. **Source** 選擇 `main` 分支
5. **Folder** 選擇 `/ (root)`
6. 點擊 **Save**

### 3. 訪問網站

等待幾分鐘後，訪問：
```
https://yourusername.github.io/programshowsystem/
```

## ⚠️ 注意事項

1. **瀏覽器相容性**
   - 建議使用 Chrome、Firefox、Safari 或 Edge
   - 需要支援 LocalStorage 和 FileReader API

2. **資料儲存**
   - 所有資料存在瀏覽器 LocalStorage
   - 清除瀏覽器資料會導致檔案遺失
   - 建議定期使用「匯出資料」功能備份

3. **檔案限制**
   - 僅支援 `.py` 檔案
   - 單檔最大 1MB
   - 總儲存空間約 5-10MB

## 🆘 遇到問題？

1. **無法上傳檔案**
   - 確認檔案是 `.py` 格式
   - 確認檔案大小不超過 1MB

2. **展示頁面沒有語法高亮**
   - 確認網路連線（使用 CDN）
   - 重新整理頁面

3. **資料遺失**
   - 檢查是否清除了瀏覽器資料
   - 如果有匯出備份，使用「匯入資料」恢復
