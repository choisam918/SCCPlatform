## 🚀 使用方式

### 立即測試
1. 直接用瀏覽器開啟 `index.html`
2. 上傳 `examples/` 目錄中的範例檔案
3. 測試所有功能

### 部署到 GitHub
1. 推送到 GitHub Repository
2. 啟用 GitHub Pages
3. 訪問 `https://username.github.io/programshowsystem/`

詳細步驟請參考 [快速開始.md](./快速開始.md)

## ⚠️ 注意事項

1. **瀏覽器相容性**
   - 需要現代瀏覽器（Chrome 60+, Firefox 55+, Safari 11+, Edge 79+）
   - 需要支援 LocalStorage 和 FileReader API

2. **儲存限制**
   - LocalStorage 約 5-10MB（依瀏覽器而定）
   - 建議定期匯出備份

3. **網路需求**
   - highlight.js 使用 CDN，需要網路連線
   - 如需離線使用，可下載 highlight.js 到本地
