# Python 程式碼執行功能說明

## 🎯 功能概述

系統現在支援在瀏覽器中直接執行 Python 程式碼並顯示執行結果！

## ✨ 如何使用

### 1. 上傳 Python 檔案
- 在 `index.html` 頁面上傳您的 Python 檔案

### 2. 生成展示頁面
- 點擊「生成展示頁面」按鈕

### 3. 執行程式碼
- 在展示頁面中，每個程式碼區塊下方都有「▶️ 執行程式碼」按鈕
- 點擊按鈕即可執行該 Python 檔案
- 執行結果會顯示在程式碼下方

### 4. 查看結果
- **標準輸出**：`print()` 的輸出會正常顯示
- **錯誤訊息**：如果有語法錯誤或執行錯誤，會以紅色顯示
- **執行完成**：如果程式執行完成但無輸出，會顯示提示訊息

## 🔧 技術實現

### Pyodide
系統使用 **Pyodide** 在瀏覽器中執行 Python 程式碼。

- **Pyodide** 是將 Python 編譯為 WebAssembly 的專案
- 可以在瀏覽器中運行完整的 Python 3.x 環境
- 支援大部分 Python 標準庫

### 執行環境
- Python 版本：3.11（透過 Pyodide）
- 支援的庫：大部分 Python 標準庫
- 不支援：檔案系統操作、網路請求等受限功能

## ⚠️ 注意事項

### 1. 首次執行
- **首次執行需要載入時間**：Pyodide 約 10MB，首次載入需要 10-30 秒
- 需要穩定的網路連線
- 載入完成後會快取，後續執行會更快

### 2. 功能限制
以下功能可能無法正常運作：
- ❌ 檔案讀寫（`open()`, `file()` 等）
- ❌ 網路請求（`urllib`, `requests` 等）
- ❌ 系統調用（`os.system()`, `subprocess` 等）
- ❌ 某些第三方庫（需要額外安裝）

### 3. 支援的功能
以下功能可以正常使用：
- ✅ 基本 Python 語法
- ✅ 資料結構（list, dict, tuple, set）
- ✅ 函數和類別
- ✅ 數學運算
- ✅ 字串處理
- ✅ 大部分標準庫（math, random, datetime 等）

## 📝 範例

### 範例 1：基本輸出
```python
print("Hello, World!")
print("這是一個測試")
```

**執行結果**：
```
Hello, World!
這是一個測試
```

### 範例 2：計算
```python
def add(a, b):
    return a + b

result = add(10, 20)
print(f"10 + 20 = {result}")
```

**執行結果**：
```
10 + 20 = 30
```

### 範例 3：錯誤處理
```python
x = 10
y = 0
result = x / y  # 會產生錯誤
```

**執行結果**：
```
ZeroDivisionError: division by zero
```

## 🚀 效能說明

- **載入時間**：首次載入約 10-30 秒（依網路速度）
- **執行速度**：比原生 Python 慢，但對於教學展示足夠
- **記憶體使用**：約 50-100MB（瀏覽器記憶體）

## 💡 使用建議

1. **教學用途**：非常適合展示 Python 程式碼的執行結果
2. **簡單程式**：適合執行簡單的計算、字串處理等
3. **複雜程式**：不建議執行需要檔案操作或網路請求的程式

## 🔍 故障排除

### 問題：點擊執行沒有反應
- 檢查網路連線
- 查看瀏覽器控制台是否有錯誤
- 等待 Pyodide 載入完成（首次需要時間）

### 問題：執行結果顯示錯誤
- 檢查 Python 程式碼是否有語法錯誤
- 確認使用的功能是否支援
- 查看錯誤訊息了解具體問題

### 問題：執行速度很慢
- 這是正常現象，Pyodide 執行速度比原生 Python 慢
- 複雜計算可能需要較長時間

---

**提示**：這個功能主要用於展示和教學，不適合執行複雜的生產環境程式碼。
