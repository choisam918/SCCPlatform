# 專案結構說明

```
programshowsystem/
│
├── index.html                 # 主頁面（上傳介面）
├── display.html               # 展示頁面
├── README.md                  # 專案說明
├── 使用說明.md                # 使用說明文檔
├── 專案結構.md                # 本檔案
│
├── .gitignore                 # Git 忽略檔案
│
├── css/                       # 樣式目錄
│   └── style.css              # 主要樣式檔案
│
├── js/                        # JavaScript 目錄
│   ├── storage.js             # LocalStorage 管理模組
│   ├── fileHandler.js        # 檔案處理模組
│   ├── main.js               # 主程式邏輯（上傳頁面）
│   └── display.js            # 展示頁面邏輯
│
├── lib/                       # 第三方庫目錄
│   └── highlight.js/          # highlight.js（程式碼高亮）

│
└── docs/                      # 文檔目錄
    ├── 系統設計建議.md
    ├── 方案對比表.md
    ├── 系統架構流程圖.md
    └── GitHub部署方案.md
```

## 檔案說明

### HTML 檔案

- **index.html**：主頁面，提供檔案上傳功能
- **display.html**：展示頁面，顯示所有上傳的 Python 程式碼

### CSS 檔案

- **css/style.css**：包含所有頁面樣式，支援響應式設計和深色主題

### JavaScript 檔案

- **js/storage.js**：負責 LocalStorage 的讀寫操作
  - `getAllFiles()` - 獲取所有檔案
  - `addFile()` - 添加檔案
  - `deleteFile()` - 刪除檔案
  - `exportData()` - 匯出資料
  - `importData()` - 匯入資料

- **js/fileHandler.js**：處理檔案上傳和操作
  - `handleFiles()` - 處理檔案上傳
  - `previewFile()` - 預覽檔案
  - `deleteFileHandler()` - 刪除檔案處理

- **js/main.js**：上傳頁面的主要邏輯
  - `initializePage()` - 頁面初始化
  - `renderFilesList()` - 渲染檔案列表
  - `generateDisplay()` - 生成展示頁面

- **js/display.js**：展示頁面的主要邏輯
  - `initializeDisplay()` - 展示頁面初始化
  - `renderDisplay()` - 渲染展示內容
  - `exportHTML()` - 匯出 HTML
  - `toggleTheme()` - 切換主題

### 配置檔案

- **.gitignore**：Git 版本控制忽略檔案清單

### 文檔檔案

- **README.md**：專案總覽和快速開始
- **使用說明.md**：詳細的使用說明
- **docs/**：系統設計文檔

## 資料流程

```
使用者上傳檔案
    ↓
fileHandler.js 處理檔案
    ↓
storage.js 儲存到 LocalStorage
    ↓
main.js 更新介面顯示
    ↓
使用者點擊「生成展示」
    ↓
跳轉到 display.html
    ↓
display.js 讀取檔案並渲染
    ↓
highlight.js 進行語法高亮
```

## 擴展建議

### 添加新功能

1. **搜尋功能**：在 `js/display.js` 中添加搜尋邏輯
2. **檔案分類**：修改 `js/storage.js` 添加分類欄位
3. **自訂主題**：在 `css/style.css` 中添加新主題
4. **匯出PDF**：使用 jsPDF 庫添加 PDF 匯出功能

### 優化建議

1. **效能優化**：大量檔案時使用虛擬滾動
2. **錯誤處理**：添加更完善的錯誤提示
3. **載入動畫**：添加檔案處理時的載入動畫
4. **鍵盤快捷鍵**：添加鍵盤快捷鍵支援

## 技術依賴

### 外部依賴

- **highlight.js**：透過 CDN 載入（版本 11.9.0）
  - 用於 Python 程式碼語法高亮

### 瀏覽器 API

- **LocalStorage API**：資料儲存
- **FileReader API**：檔案讀取
- **Blob API**：檔案下載
